Audio
*****

The ``ark-audio`` allows the user to record audio using the current device microphone.

Example
=======

    .. code:: html

        <ark-audio></ark-audio

Attributes
----------

+--------+---------+------------------------------------------------------------+
|  Name  | Default |                        Description                         |
+========+=========+============================================================+
| status | -       | Different status (idle,recording,done)  adds automatically |
+--------+---------+------------------------------------------------------------+

----------
