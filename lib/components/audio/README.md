Audio
=====

`ark-audio` allows the user to record audio using the current microphone device.

## Example

``` html
  <ark-audio></ark-audio>
```

## Attributes

|  Name  | Default |                        Description                         |
| :----: | :-----: | :--------------------------------------------------------: |
| status |    -    | Different status (idle,recording,done)  adds automatically |

## Properties

|  Name  | Default |                        Description                         |
| :----: | :-----: | :--------------------------------------------------------: |
| status |    -    | Different status (idle,recording,done)  adds automatically |
